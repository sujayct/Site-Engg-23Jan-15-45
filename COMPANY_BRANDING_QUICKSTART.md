# Company Branding - Quick Start Guide

## Accessing Company Profile

1. Login as Admin
2. Go to Admin Dashboard
3. Click **"Company Profile"** tab
4. Configure your branding settings

## Configuration Fields

### Required Fields
- ✅ Company Name
- ✅ Brand Name
- ✅ Primary Color
- ✅ Secondary Color
- ✅ Support Email
- ✅ Contact Number
- ✅ Address

### Optional Fields
- Logo (PNG/JPG, max 5MB)

## Quick Steps

1. **Enter Company Details**
   - Fill in company name and brand name
   - These appear on login page and headers

2. **Upload Logo (Optional)**
   - Click "Choose File"
   - Select PNG or JPG (max 5MB)
   - Logo appears immediately in preview

3. **Choose Colors**
   - Use color pickers or enter hex codes
   - Primary: Main brand color
   - Secondary: Accent color

4. **Add Contact Info**
   - Support email for email footers
   - Phone number for contact
   - Full company address

5. **Preview Changes**
   - Click "Preview" to see how it looks
   - Review colors, logo, and layout

6. **Save**
   - Click "Save Changes"
   - Changes apply immediately

## Where Branding Appears

- ✅ Login page (logo, colors, brand name)
- ✅ Web header (logo, brand name)
- ✅ Mobile headers (logo, colors)
- ✅ Email reports (logo, colors, footer)

## Tips

- Use your official company logo for professional appearance
- Choose high-contrast colors for readability
- Hex format required for colors (e.g., #2563eb)
- Keep file sizes reasonable for faster loading
- Test preview before saving

## Default Demo Values

If you haven't configured branding yet:
- Brand Name: "Site Engineer"
- Colors: Professional blue (#2563eb / #1e40af)
- Default contact info shown

## Need Help?

- Check form validation messages
- Use preview to test before saving
- Ensure you're logged in as Admin
- Contact support if issues persist

**That's it!** Your branding will now appear consistently across the entire platform.
