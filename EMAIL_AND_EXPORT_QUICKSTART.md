# Email Notifications & Export - Quick Start Guide

## Email Notifications

### For Engineers

**No Action Required!**

When you:
- ✅ Check in at a site
- ✅ Submit a daily report
- ✅ Request leave

**What Happens:**
- Your assigned client automatically receives an email
- Email includes all relevant details
- Professional formatting
- Completely automatic

**Note:** Make sure your profile has an active client assignment for notifications to work.

---

### For Clients

**What to Expect:**

You will receive automatic email notifications when engineers:

1. **Check In**
   - Date and time
   - Location
   - Status

2. **Submit Daily Reports**
   - Work description
   - Hours worked
   - Issues faced
   - Materials used

3. **Request Leave**
   - Start and end dates
   - Reason
   - Status (Pending/Approved/Rejected)

**Email Format:**
- Professional HTML design
- Color-coded by notification type
- All information clearly displayed
- Automated from Engineer Management System

**No Spam:**
- Only receive updates for engineers assigned to you
- Real, meaningful updates only
- No extra emails or promotions

## Report Export

### For HR

**Where to Export:**

1. **HR Dashboard → Attendance Tab**
   - Export daily attendance
   - Click "Export CSV" button

2. **HR Dashboard → Reports Tab**
   - Export daily reports
   - Click "Export CSV" button

3. **HR Dashboard → Enterprise Reports Tab**

   **Daily Attendance Register:**
   - Select date
   - Click "Export CSV"
   - Get: Engineer, Status, Check-in/out, Hours

   **Weekly Engineer Summary:**
   - Select date range (start - end)
   - Click "Export CSV"
   - Get: Present/Absent/Leave days, Hours

   **Monthly Client Report:**
   - Select month
   - Click "Export CSV"
   - Get: Client-wise engineer activity

   **Payroll Export:**
   - Select month
   - Click "Export CSV"
   - Get: Working days, Hours, Leave, Overtime

---

### Opening CSV Files

**Excel:**
1. Download CSV
2. Open Microsoft Excel
3. File → Open → Select CSV file
4. Done!

**Google Sheets:**
1. Download CSV
2. Go to Google Sheets
3. File → Import → Upload CSV
4. Done!

**All Data Included:**
- Properly formatted
- Headers included
- Ready for analysis

## Tips

### Email Notifications

- **Engineers:** Make sure you have an active client assignment
- **Clients:** Check your spam folder if emails are missing
- **Admin:** Verify client email addresses are correct in profiles

### Export

- **File Format:** CSV (opens in Excel, Google Sheets)
- **File Name:** Includes date/range automatically
- **Best Practice:** Export regularly for records
- **Archive:** Save exports for historical data

## Troubleshooting

### Not Receiving Emails?

1. Check engineer has active assignment
2. Verify client email in profile
3. Check spam/junk folder
4. Contact system admin

### Export Not Working?

1. Ensure data exists for selected date/range
2. Allow browser to download files
3. Try different browser if issue persists
4. Contact system admin
