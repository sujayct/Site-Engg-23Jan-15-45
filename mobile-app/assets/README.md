# Assets Directory

Place your app icons and splash screens here:

## Required Assets

1. **icon.png** (1024x1024)
   - Main app icon
   - Used for both iOS and Android

2. **splash.png** (1284x2778)
   - Splash screen shown on app launch
   - Should match your brand colors

3. **adaptive-icon.png** (1024x1024)
   - Android adaptive icon
   - Should work with various shapes (circle, square, rounded)

4. **favicon.png** (48x48)
   - Web version favicon
   - Optional if not building for web

## Notes

- All images should be PNG format
- Use transparent backgrounds where appropriate
- Ensure icons are clear at small sizes
- Test on both light and dark backgrounds
